# axx-cli
> https://github.com/gaosife/axx-cli


## Install

```sh
    npm install axx-cli -g --registry=https://registry.npm.taobao.org
```

## Command
### init<font color=red >（第一步，在项目目录中执行）</font>

```sh
    axx build init
```

### service

```sh
    axx build service <option>
```

```
Options:
    -b, --build  启动编译
    -d, --dev    启动本地调试
```

### add

```sh
    axx build add <npm package name>
```
