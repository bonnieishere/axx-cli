# axx-cli
> https://github.com/gaosife/axx-cli


## Install

```sh
    npm install axx-cli
```

## Command
### init

```
	axx build init <option>
```
```
Options:
    -b, --begin   初始化一个空项目（暂无）
    -c, --config  初始化配置文件
    -s, --start   执行配置文件
```
### service
```
	axx build service <option>
```
```
Options:
    -b, --build  启动编译
    -d, --dev    启动本地调试
```
### add
```
	axx build add <npm package name>
```
